# Wallpaper Module

🚧 **Coming Soon**

This module will enable natural-language control of Windows wallpapers:

- **Desktop wallpaper** — set from a file path or URL
- **Lock screen** — customize lock screen image
- **Slideshow** — configure a rotating wallpaper slideshow
- **Spotlight** — toggle Windows Spotlight

## Planned Approach

PowerShell scripts using `SystemParametersInfo` and registry settings.
