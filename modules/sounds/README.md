# Sounds Module

🚧 **Coming Soon**

This module will enable natural-language control of Windows system sounds:

- **Sound scheme** — switch between installed sound schemes
- **Individual events** — change sounds for notifications, errors, etc.
- **Volume** — adjust system or app-specific volume

## Planned Approach

PowerShell scripts that modify `HKCU:\AppEvents\Schemes` registry keys.
